Projects

Utilizing Single-Record Data to Improve Suicide Risk Prediction in EHR data

We innovated an integrative semi-supervised learning framework to improve suicide risk prediction by leveraging the information within the single-record data in Electronical Health Records (EHR), by combining supervised learning component (a latent-variable model linking historical records to multivariate outcomes) and unsupervised learning component (a low-rank approximation of the multivariate single-record outcomes) via the shared loading information.

Imputation of Missing Value in Electronical Health Records (EHR) Data

We proposed a framework for simultaneous imputation of missing covariates and prediction of outcomes, which incorporated the interrelationships among multivariate outcomes to enhance model efficacy, and regularization methods were implemented for high-dimensional variable selection.
